Este projeto foi desenvolvido utilizando React e Vite como ferramentas de desenvolvimento. Para executar o projeto, é necessário ter o Node.js instalado em sua máquina.

Para iniciar o projeto, abra um terminal na pasta raiz do projeto e execute o comando npm install para instalar as dependências do projeto.

Para executar o projeto em modo de desenvolvimento, utilize o comando npm run dev. O projeto será executado no endereço http://localhost:3000.

Para construir o projeto para produção, execute o comando npm run build. O projeto será construído na pasta dist na raiz do projeto.

Para visualizar a versão de produção do projeto, execute o comando npm run preview. O projeto será executado no endereço http://localhost:5000.

Bibliotecas utilizadas:

antd: biblioteca de componentes de interface de usuário para React.
axios: biblioteca para fazer requisições HTTP.
react: biblioteca para criar interfaces de usuário.
react-dom: biblioteca para manipulação do DOM no React.
react-router-dom: biblioteca de roteamento para React.
styled-components: biblioteca para criar estilos para componentes React utilizando CSS-in-JS.
DevDependencies:

@testing-library/react: biblioteca para testes unitários de interfaces de usuário em React.
@types/react: biblioteca para definições de tipos para o React.
@types/react-dom: biblioteca para definições de tipos para o react-dom.
@vitejs/plugin-react: plugin para integrar React com Vite.
vite: ferramenta de desenvolvimento rápido para projetos web baseados em JavaScript.
